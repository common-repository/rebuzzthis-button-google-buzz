== Description ==

= What this plugin does =

The RebuzzThis WordPress plugin allows you to instantly add a RebuzzThis Google Buzz button to your blog within minutes to let readers share your posts on Google Buzz. Easily choose what style button you want, where to align it, and even what pages it can/can't show up on.

= Features: =

* Choose from 9 different styles of buttons.
* Buttons can be hidden on posts, pages, archives, etc.
* Align the button to display in different areas of your posts and pages. 

== Installation ==

1. Upload the plugin folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in wordPress.
3. To customize Click Settings -> RebuzzThis.

== Frequently Asked Questions ==

= Which version of WordPress is supported? =

WordPress 2.5.x to 2.9.x is currently supported. This plugin has not been tested on any versions before 2.5.
